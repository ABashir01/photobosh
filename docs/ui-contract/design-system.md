# Design System

## Tokens

- Page background: warm off-white
- Primary ink: near-black
- Accent: also near-black; emphasis should come from weight and contrast, not color
- Border style: thin soft rules
- Radius: generous only where it helps frame the booth or strip

## Typography

- Headline scale should be large and compact, with strong black-and-white contrast.
- Secondary metadata should use mono or mono-like styling.
- Avoid app-dashboard label styling.

## Layout Rules

- Mobile is the primary composition target.
- Main content width should stay narrow and deliberate on larger screens rather than becoming a dashboard.
- The live booth should dominate the waiting-lobby and capture steps.
- Theme selection should look like a strip browser, not a settings form.

## Anti-Drift Rules

- Do not reintroduce broad panel grids.
- Do not rely on gradients for product identity.
- Do not split the waiting lobby into multiple competing columns on desktop.
